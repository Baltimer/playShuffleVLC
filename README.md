#Play Shuffle

##Presentación

Programa que pide una biblioteca de canciones y crea una Play List aleatoria, la cual se reproducirá en el reproductor __VLC__
(en este caso hecho para S.O. Windows).

##Dificultades

Las funciones realizadas en clase han salido de manera sencilla y ágil siguiendo las indicaciones
del profesor, pero a la hora de realizar las otras funciones nos hemos encontrado con los
siguientes problemas:
- Al realizar la función imprimirPlayList nos hemos dado cuenta que era poco eficiente y hemos necesitado un tiempo 
para encontrar una solución más eficiente por nuestra cuenta.
- Al querer realizar algunas funciones del programa, no entendíamos muy bien los parámetros que estaban especificados 
en las funciones predefinidas del programa dado por el profesor.
